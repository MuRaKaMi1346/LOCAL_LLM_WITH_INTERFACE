ทำตามขั้นตอนด้านล่างเพื่อให้บอทรับข้อความจาก LINE ได้จริง

1
เปิด ngrok ใน launcher

2
วาง URL ใน LINE Developers Console
เปิด developers.line.biz → เลือก Channel → Messaging API → Webhook settings
วาง URL ในช่อง Webhook URL แล้วกด Verify
3
เปิด Use webhook
ตรวจสอบให้แน่ใจว่าเปิด "Use webhook" และปิด "Auto-reply messages" ใน LINE Console
หลังจากนั้นลอง Add บอทเป็นเพื่อนแล้วส่งข้อความทดสอบ

🔍 RAG & การสนทนา

RAG (Retrieval-Augmented Generation) คือระบบที่ช่วยให้บอทค้นหาข้อมูลจากเอกสารที่คุณอัพโหลดก่อนตอบ ทำให้ตอบแม่นยำขึ้น

จำนวน chunk อ้างอิงสูงสุดต่อคำถาม — ค่าที่แนะนำ: 10
ความยาวของแต่ละส่วนเอกสาร (ตัวอักษร) — ค่าที่แนะนำ: 1200
ส่วนทับซ้อนระหว่าง chunk ช่วยให้ไม่ขาดตอน — ค่าที่แนะนำ: 150
จำนวนรอบสนทนาที่บอทจำได้ต่อผู้ใช้ — ค่าที่แนะนำ: 8
